# Retail Sales Analytics Dashboard

## Overview
An interactive retail sales dashboard built using LibreOffice Calc, analyzing 1000+ transaction records across regions, product categories, and shipping modes.

## Tools Used
- LibreOffice Calc
- Pivot Tables
- Charts (Bar, Column, Pie)
- Excel formulas

## Dataset
1000 rows, 19 columns including Order Date, Region, Category, Sub-Category, Sales, Profit, Ship Mode, and more.

## Key Features
- 5 Pivot Tables (Region, Category, Sub-Category, Top Products, Ship Mode)
- 5 interactive charts
- KPI cards: Total Sales, Total Profit, Total Orders, Profit Margin
- Clean dashboard layout for business insights

## Key Insights
- Total Sales: ₹2,53,98,597.99
- Total Profit: ₹70,15,021.50
- Overall Profit Margin: 27.62%
- Top Sub-Category by Sales: Tables (₹48,33,908.57)
- Total Orders: 1000
